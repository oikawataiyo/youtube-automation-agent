# 風の谷のナウシカ 開拓史編

## 1行サマリ
風の谷は自然の理想郷ではなく、祖先の犠牲とインフラで成立した人工の生存装置である。

## 岡田の核心テーゼ (なぜ刺さるか)
岡田の読みでは、『ナウシカ』の強さは「設定がある」ことではなく、設定が画面の物体、地形、音、動線として見えることにある。冒頭の廃村、胞子、ミイラ、壊れる人形、風車、砂よけ棚、貯水池は、説明台詞ではなく観察可能な証拠として世界の歴史を語る。

風の谷は、明るい風車やブドウ畑のある牧歌的な土地に見える。しかし実際には、腐海の毒、砂塵、セラミック片、水不足、風向きの変化に常時さらされる危険地帯であり、怠け者が生き残れないほど過酷な共同体として設計されている。

この世界観構築の要点は、画面に出る構造物を「美術」ではなく「生存のための機械」として読むことにある。虫除けの塔、砂よけ棚、用水風車、石積みの水道、城の位置関係は、すべて風の谷の開拓史を語る記録媒体になっている。

視聴者は最初、腐海と虫を絶対悪として受け取る。しかし物語が進むにつれて、その悪に見えたものの意味が反転していく。岡田はこの感情変化を、冒頭で小さな日常を壊す恐怖を十分に見せておくからこそ成立する構造として捉えている。

## 岡田 frame (独自命名分析)
- **ブラナウシカ**: 風の谷を旅行記のように歩き、地形と設備から世界史を読む方法 — ユパの入村ルートが開拓史の導線になる — ファンタジー世界は観光動線で説明できる。
- **画面からの読み取り**: 絵コンテや台詞ではなく、画面にある形・素材・位置から設定を復元する方法 — 石造りの塔から鉄の不在を読む — 背景美術を情報源として設計する。
- **ポストセラミック時代**: 便利すぎる素材文明が崩壊し、再生不能な残骸だけが残った時代設定 — 砂に見えるものをセラミック片として読む — 文明崩壊は「失った素材」で描く。
- **支えるに足りる共同体**: 苦労してでもそこに住む価値がある共同体という宮崎作品の大テーマ — 風の谷は毒の中でも守る価値を持つ — 共同体の魅力は快適さでなく倫理で作る。
- **自然に出来たものは何一つない**: 美しい風景を人工の生存装置として読む frame — 風車、土、水路、果樹園がすべて祖先の作業結果 — 世界観は労働履歴として作る。
- **絶対悪の反転構造**: 冒頭で悪に見せた存在を終盤で美しく見せる感情設計 — 腐海と虫が恐怖から浄化へ反転する — 敵対物の意味を段階的に組み替える。
- **インフラで語る開拓史**: 歴史説明を年表でなく装置の配置で語る方法 — 砂よけ棚の多重構造が何世代もの工事を示す — 架空都市は上下水道と防災で語る。
- **一瞬の小道具で家庭を見せる**: 大事件を小さな生活物で感情化する方法 — 王女の人形が滅亡を個人化する — 世界崩壊は玩具や食器で描く。
- **風の谷の二つの海**: 同じ「海」を安全側と腐海側に分け、地形の緊張を作る方法 — 谷は二つの海に挟まれた狭い生存圏 — 地名を機能差で分割する。
- **背景美術の機能化**: 美しい背景を装飾で終わらせず、必ず働きを持たせる方法 — 風車は絵ではなく水と防御の機械 — 絵作りは機能設計から始める。
