# 95％の悩みを解決する思考方法

## 1行サマリ
悩みを解くのではなく、書き出し・分類・距離化・五つの対処法で扱える形にするメタ講義。

## 岡田の核心テーゼ (なぜ刺さるか)
この回は岡田自身のメタ frame 集である。核心は、悩みとは「問題が頭の中で絡まり、ワークスペースを超えて動けなくなる状態」であり、まず解決するのではなく、外へ出し、分類し、扱える問題へ戻すことにある。
岡田は悩みを、テーブル、2トン車、ゴミ屋敷、色鉛筆、カメラを引く、といった具象物で説明する。心理状態を見える配置に変えるため、視聴者は自分の混乱を責めずに操作対象として見られる。
さらに重要なのは、解決だけが答えではないという五つの対処法である。解決する、逃げる、保存する、忘れる、共有する。悩み相談は答えを出す仕事ではなく、考える順番と視点を提案する仕事として再定義される。
moyo 向けには最も直接的に使える。認知行動療法的な外在化、ジャーナリング、問題分解、セルフコンパッション、メタ認知を、岡田式の比喩でエンタメ化できる。特に「面白がる」は、苦痛を客観視へ変える強い frame である。

## 演出技法 (具体)
### キャラ造形 / 表情 / シンボル
- (該当言及なし: 映画的キャラ造形ではない)
- 自己キャラ化: 岡田自身を「悩みは少ないが考え事は多い人」として置く → 方法論に信頼と癖が出る
- 相談者の扱い: 壮絶シングルマザー、携帯盗み見の女性を「本当に聞きたいことがズレている人」として読む → 悩みを物語化する
- シンボル: テーブル、2トン車、色鉛筆、カメラ上昇 → 抽象的な思考操作を視覚化する

### 色 / 光 / 構図 / カット
- (該当言及なし: 色彩・カットの映像作品分析はない)
- ホワイトボード図解: 頭の中のモヤモヤをテーブル上の物として描く → 悩みの外在化を視覚化する
- 色分け: 相談文を今・過去・未来に分ける → 問題の優先順位を見せる
- カメラを引く比喩: 主観から離れ、上から自分を見る → 面白がるための視点移動になる

### 音 / 間 / リズム
- (該当言及なし: 音響演出への直接言及はない)
- 体験談から理論へ: 自分の悩みを書き出す話から一般理論へ進む → 抽象化が自然
- 相談事例で検証: 理論を出した後、シングルマザー相談で実演する → 信頼が上がる
- 五つの対処法で締める: 解決一択から複数選択へ広げる → 心理的な出口が増える

## 物語構造の技法
- **起承転結の崩し**: 悩み相談なのに解決をゴールにしない。まず整理、距離化、扱い方の選択に置き換える。
- **伏線**: 冒頭の書き出し癖が、後の相談文色分けと五つの対処法に回収される。
- **メタファー**: 頭のワークスペース、2トン車、テーブル、ゴミ屋敷、カメラを引く。
- **メタ構造 (作中作・劇中の視点)**: 悩み相談を読みながら、相談文そのものの構造を分析する。
- **ダブルミーニング (表層と深層)**: 「何を答えるか」ではなく「本当は何に悩んでいるか」を読む。

## 視聴者心理操作
- **期待裏切り**: 相談には答えを出すべきという期待を外し、整理するだけでも効くと示す。
- **共感誘導**: 時間がない、何を話せばいいのか、腰が痛いなど小さな不安を列挙し入口を作る。
- **不快感制御**: 深刻な相談を、色分けや分類という作業に変えて圧を下げる。
- **解釈余白 / 知識補完欲求**: 相談者の本当の問いを読解するため、視聴者も一緒に推理する。

## 岡田 frame (独自命名分析)
- **頭のワークスペース**: 悩みは処理容量を超えた問題の混線 — 頭の中のモヤモヤ — 認知負荷の説明に使える
- **テーブルに乗せる**: 書き出して外在化する — 紙に移せば覚えなくてよい — ジャーナリング解説に使える
- **仕分け**: 問題をジャンル・時間軸・関係有無で分ける — 相談文の色分け — 問題解決動画に使える
- **五つの対処法**: 解決する、逃げる、保存する、忘れる、共有する — 解決以外の選択肢 — 悩み相談の基本型に使える
- **面白がる**: カメラを引き、主観から離れて状況を観察する — 腰痛や携帯盗み見相談 — 苦痛のメタ認知に使える
- **聞かれたことに答えない**: 表面の質問ではなく奥の不安へ答える — 携帯盗み見の相談 — カウンセリング的解説に使える

## 応用 hint (この記事から学べる転用)
- 心理学解説動画なら: 悩みを「解決すべき敵」ではなく「分類前の材料」として扱う。
- 抽象テーマを扱うなら: ワークスペース、テーブル、色鉛筆のように操作できる比喩へ変換する。
- 短尺なら: 「紙に書く→今関係あるものだけ残す→五つの対処法から選ぶ」の3ステップでまとめる。
- moyo向けなら: 不安の外在化、問題分解、メタ認知、逃げる選択肢の肯定を中心にする。
- moyo向けなら: 「聞かれたこと」ではなく「本当の不安」を探す読解型の悩み相談にする。

## 応用マトリクス
### 応用 01: 冒頭で結論を言い切る
- 使う場面: 長い書籍解説を短時間で入れる時
- 構成手順: 結論を三語程度に圧縮し、そのあと数字・逸話・反論で広げる
- 注意点: 結論だけで終わらせず、視聴者が自分に関係あると感じる例を置く
- 短尺化: 冒頭10秒で「これは何の話か」を言う

### 応用 02: 抽象語を身体化する
- 使う場面: 倫理・社会制度・脳科学など目に見えない話題
- 構成手順: 車、テーブル、パチンコ屋、消防士など触れる比喩に置き換える
- 注意点: 比喩が面白すぎて本題から離れないよう戻しの一文を入れる
- 短尺化: 比喩一つ、結論一つに絞る

### 応用 03: 反論を先に出す
- 使う場面: 視聴者がすでに反対意見を持つテーマ
- 構成手順: 「こう思う人もいる」と代弁してからデータや構造で返す
- 注意点: 相手を馬鹿にせず、反論の魅力を一度認める
- 短尺化: 反論を一行テロップ化する

### 応用 04: 不快情報に笑いの逃げ道を作る
- 使う場面: 人体実験、差別、依存など暗い話題
- 構成手順: えげつない事実の直後に語り手の驚きや比喩を挟む
- 注意点: 被害そのものを笑わず、構造の異常さを笑う
- 短尺化: 不快な絵の前後に一拍置く

### 応用 05: 自分の読み方として提示する
- 使う場面: 本の正解要約ではなく解釈を語る時
- 構成手順: 「岡田はこう読んだ」と主観の足場を明示する
- 注意点: 原著の断定と自分の推測を混ぜない
- 短尺化: 「この動画ではここだけ読む」と限定する

### 応用 06: 段階構造で理解を支える
- 使う場面: 複数の論点を扱う長尺解説
- 構成手順: 3要素、5手段、8トピックのように番号で棚を作る
- 注意点: 番号が増えすぎると記憶できないため途中で要約する
- 短尺化: 3つ以下に圧縮する

### 応用 07: 日常例で倫理を下ろす
- 使う場面: 政治哲学や社会思想を扱う時
- 構成手順: 婚活、学校、仕事、スマホ、家族など自分事に接続する
- 注意点: 例が俗っぽくても論点の射程を失わない
- 短尺化: 日常例から始めて最後に概念名を出す

### 応用 08: 恐怖を未来予測に変える
- 使う場面: 技術・社会変化のリスクを語る時
- 構成手順: 過去の惨事、現在の数字、未来の帰結を一直線に並べる
- 注意点: 煽りで終えず、どう見るべきかを提示する
- 短尺化: 過去-現在-未来を三画面で切る

### 応用 09: 語り手の困りを見せる
- 使う場面: 扱いにくいテーマを始める時
- 構成手順: 「今日は難しい」「面白くする方法がない」と先に言う
- 注意点: 免責で逃げず、その後の構成を明確にする
- 短尺化: 難しさの宣言は一文だけにする

### 応用 10: 最後に使える手順へ戻す
- 使う場面: 抽象度の高い分析を終える時
- 構成手順: 明日使える問い、分類、制限、観察法に落とす
- 注意点: 精神論ではなく行動単位で書く
- 短尺化: 一つのチェックリストにする
