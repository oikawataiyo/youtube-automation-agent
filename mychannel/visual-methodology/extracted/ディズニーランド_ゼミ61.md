# ディズニーランド

## 1行サマリ
ディズニーランドは夢の国である前に、滞在時間、錯覚、運営費を設計した没入体験装置である。

## 岡田の核心テーゼ (なぜ刺さるか)
岡田のディズニーランド論の核心は、「夢と魔法」を支えるものが徹底した打算であるという点にある。入場料、スタッフ配置、酒を売らない判断、列の映像、建物の縮尺、通路の曲げ方、配色、土地取得、不動産まで、体験の美しさは運営と収益の設計に支えられている。

ディズニーランドは映像作品ではないが、映画的な演出原理で作られている。ウォルト・ディズニーは建築家ではなく映画屋であり、セットを大きく見せる遠近法、色彩による距離感、視界を遮る通路設計、中心の城へ向かわせる構図を空間に転用した。

体験設計として重要なのは、顧客満足度を「全部回れる」ではなく「滞在時間内に80%回れる」程度に調整する発想である。完全制覇させず、未体験を残すことで再訪欲求を作る。同時に、歩きにくさや見通しの悪さを不満ではなく広さの錯覚に変えている。

さらに岡田は、テーマパークの成功条件として「いかにスタッフを減らすか」「固定費をどう制御するか」を強調する。列のモニターやライドの自動化は豪華演出に見えるが、実際には人件費とトラブルを減らすための初期投資である。没入体験は、感情設計とコスト設計が一致した時に強くなる。

## 岡田 frame (独自命名分析)
- **夢と魔法と打算の国**: 夢の体験を支える経営・運営・錯覚の設計を読む frame — 値上げ、列映像、縮尺、土地取得が全部つながる — 感動体験は打算を隠さず設計する。
- **顧客満足度最大値化**: 平均滞在時間と回遊率から面積を決める frame — 5.5時間で80%回れる広さにする — 全部見せず、満足と未完了を両立する。
- **いかにスタッフを減らすか**: 自動化と初期投資で運営費を抑える frame — 列映像やライド自動化が人件費を減らす — サービス設計は固定費から逆算する。
- **映画セット空間化**: 映画美術の錯覚を歩ける建築に転用する frame — 縮尺、色、くぼみ、曲線通路で広く高く見せる — 実空間もカメラなしのセットとして設計できる。
- **80%回遊設計**: 全部体験させないことで再訪欲求を作る frame — 滞在内に園内の大半を見せつつ残す — 教材や展示にも余白を残す。
- **待機時間の物語化**: 行列を本編前の導入に変える frame — モニターやセットで待ち時間を演出する — ユーザーの待機はコンテンツ化できる。
- **酒を売らない体験安全**: 収益源を捨てて感情トラブルを減らす frame — 園内の大声や喧嘩を予防する — 体験設計では禁止が価値を作る。
- **周辺土地までの没入管理**: パーク外の安っぽさを排除する frame — アナハイムの失敗からフロリダの巨大買収へ — 没入は境界外の景色まで含む。
- **ローカル運営の最適化**: 本国マニュアルを観客体験に合わせて調整する frame — 東京では身長や運営実態を柔軟化する — コピーではなく地域習慣で設計する。
- **テーマパーク不動産モデル**: シンボル施設で周辺価値を上げる frame — フロリダやハウステンボスの対比 — 体験拠点は土地価値戦略にもなる。
