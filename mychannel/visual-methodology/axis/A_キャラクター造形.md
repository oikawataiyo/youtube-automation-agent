# A. キャラクター造形

## 1. 軸の定義
この軸は、人物や非人間キャラを「どう見える存在として設計するか」を扱う。顔、身体、表情、目線、向き、服装、小道具、名前、立ち位置、所属状態を通じて、台詞で説明しない人物像を作る技法を抽出する。
岡田メソッドでは、キャラは内面心理の容器ではなく、世界観・倫理・歴史・観客の立場を画面に固定する装置である。映画作品では表情や身体所作を中心に、書籍解説では抽象概念を具体的人物や擬人化された敵へ落とす方法として扱う。

## 2. 中核原則 (Top-Level Principles)

### 原則 1: 表情は感情の答えではなく、観客に読ませる空欄である
キャラの顔に泣き顔や怒り顔をそのまま貼らず、無表情、作り笑顔、視線回避、照れ笑いのような曖昧な表情を置く。観客は前後のカット、行動後の痕跡、状況の矛盾から感情を補完するため、人物像は一義的な説明より深く残る。 `[出典: 火垂るの墓_UG226, 火垂るの墓_ゼミ226, もののけ姫_UG, ブレードランナー_UG, となりのトトロ_UG]`

### 原則 2: 主人公は強さよりも不安定な所属で異界に入る
千尋は転校前・引っ越し途中・家庭内で目線をもらえない子として始まり、清太は死後も現在に縛られ、アシタカは共同体から送り出される。主人公がどこにも完全に属していない状態を作ると、異界侵入、追放、転落、反復の説得力が生まれる。 `[出典: 千と千尋_UG306_前編, 千と千尋_UG307_後編, 火垂るの墓_UG226, もののけ姫_UG]`

### 原則 3: 善悪はキャラ単体ではなく、役割の二重性として作る
湯婆婆は暴君だが契約を守る経営者であり、清太は被害者だが残酷さも持ち、ジョーカーはかわいそうな弱者であり怖い加害者でもある。善人・悪人を固定せず、愛情と支配、救済と管理、弱さと暴力を同居させることで、観客の安全な倫理判断を揺らす。 `[出典: 千と千尋_UG307_後編, 火垂るの墓_ゼミ226, ジョーカー_UG, OnYourMark_UG311, 世にも奇妙な人体実験_UG]`

### 原則 4: 非人間キャラは「かわいい外見」と「由来の怖さ」を分離して設計する
トトロは商品的にはかわいいが、目を合わせない古代神であり、猫バスは面白いが化け猫であり、翼の少女は天使に見えるが人工生命または危険な異常個体でもある。非人間キャラは見た目の安心と存在論的な不安を分けるほど、子供向け表層と大人向け深層を同時に保てる。 `[出典: となりのトトロ_UG, OnYourMark_UG311, ブレードランナー_UG, かぐや姫の物語_UG231]`

### 原則 5: 小物・衣装・身体の一点で、言えない設定を背負わせる
ドロップ缶、カヤの小刀、ピエロメイク、レプリカントの目、ロイの釘、王女の人形、ミッキーの身長比率のように、キャラの情報は説明台詞ではなく物に預ける。小物は人物の過去、階層、関係、死、性、ブランド、倫理を一瞬で画面に固定する。 `[出典: 火垂るの墓_UG226, もののけ姫_UG, ジョーカー_UG, ブレードランナー_UG, ナウシカ_開拓史編_UG]`

### 原則 6: 抽象概念は顔つきのある人物配置へ変換する
書籍解説では、飢餓・伝染病・戦争を敵にし、成功者と敗者を能力主義の登場人物にし、スマホをドラッグやパチンコ屋に擬人化する。抽象テーマを「誰が何に苦しめられているか」の人物関係へ落とすと、思想や科学の話でも映像的なキャラクター造形として機能する。 `[出典: ホモデウス_UG, 能力主義_ゼミ404, スマホ脳_UG, 思考方法論_ゼミ315, 世にも奇妙な人体実験_UG]`

### 原則 7: キャラらしさは比率・動き・運用まで含めて管理する
ディズニーランドのキャラクター造形では、ミッキー単体の見た目だけでなく、相対身長、専用の動き、接触距離、中の人の条件、現場スタッフの判断権までがブランド体験に含まれる。キャラは絵柄ではなく、観客が出会うたびに同じ存在として信じられる運用システムで維持される。 `[出典: ディズニーランド_ゼミ61]`

## 4. 岡田 frame 集 (この軸で岡田が独自命名した分析 frame)

- **クレショフ読み / クレショフ効果**: 表情そのものではなく、前後のカットで観客が感情を読んでいると見る frame。適用作品は `火垂るの墓_UG226`, `火垂るの墓_ゼミ226`。汎用化するには、顔を無表情に保ち、死体・食物・残骸などの別カットで心理を発生させる。
- **文学アニメ**: 何を考えているかわからない人物を置き、観客に答えを出させる frame。適用作品は `火垂るの墓_UG226`, `火垂るの墓_ゼミ226`。汎用化するには、内面説明を削り、複数の行動証拠を並べる。
- **宙ぶらりん frame**: 所属が不安定な人物だけが異界に入れると見る frame。適用作品は `千と千尋_UG306_前編`。汎用化するには、引っ越し、転校、家庭不和、職場不適応などを導入条件にする。
- **セリフより絵 frame**: 台詞よりカット、姿勢、構図、出来事の矛盾を優先する frame。適用作品は `千と千尋_UG307_後編`。汎用化するには、人物の秘密を説明台詞でなく目線・手・距離に置く。
- **死んだ兄 frame**: ハクを千尋のために死んだ兄と読み、母の冷たさや列車行きを統合する frame。適用作品は `千と千尋_UG307_後編`。汎用化するには、画面外の死者を主人公の行動原理として痕跡化する。
- **上手下手の向き frame**: 人物の向きで、状況に流される側か止める側かを表す frame。適用作品は `OnYourMark_UG311`。汎用化するには、転機でキャラの向きや立ち位置を反転させる。
- **飛べない天使 frame**: 翼があるのに飛べる身体ではない少女を置き、シンボルと現実の落差で悲劇を作る frame。適用作品は `OnYourMark_UG311`。汎用化するには、希望記号に機能不全の身体条件を与える。
- **反射で助けたロイ説**: 救助を人間性でなくプログラム反射とも読ませる frame。適用作品は `ブレードランナー_UG`。汎用化するには、善行の直後に照れ笑いなど多義的な表情を置く。
- **フィリップ・K・ディック的倫理**: 非人間を傷つけると、対象ではなく扱う側の魂が汚れると見る frame。適用作品は `ブレードランナー_UG`。汎用化するには、人工物や架空キャラへの扱いを人間側の変質として描く。
- **目の反射判定**: 人工物の正体を目の光で示す frame。適用作品は `ブレードランナー_UG`。汎用化するには、正体の説明を小さな視覚サインへ移す。
- **アシタカせっ記フレーム**: タイトルに反してサンではなくアシタカ伝説として読む frame。適用作品は `もののけ姫_UG`。汎用化するには、タイトルや表層ヒロインに騙されず視点人物の変化を追う。
- **呪い=怪物化説**: 呪いを死亡予告でなく主人公が祟り神になる危険と読む frame。適用作品は `もののけ姫_UG`。汎用化するには、能力を強化と破滅の二重サインにする。
- **チャームの能力 / 重力場としての魅力**: 魅力を心理ではなく周囲の判断を歪ませる力として扱う frame。適用作品は `かぐや姫の物語_UG231`。汎用化するには、本人の意志と本人が周囲へ及ぼす作用を分ける。
- **三種類のおばけ分類**: 猫バス、ススワタリ、トトロを化け猫・付喪神・古代神として分ける frame。適用作品は `となりのトトロ_UG`。汎用化するには、非人間キャラの由来ごとに行動原理を変える。
- **見てはいけない神**: 神は見えたら祝福ではなく危険だとする frame。適用作品は `となりのトトロ_UG`。汎用化するには、目線を合わせない、言葉を話さない、住処に禁忌を置く。
- **悪のインフルエンサー**: 悪を支配するのでなく、周囲に流行させる存在として描く frame。適用作品は `ジョーカー_UG`。汎用化するには、カリスマの魅力より模倣と拡散の構造を描く。
- **笑いとは攻撃の裏返し**: 笑いを幸福ではなく攻撃性や関係性の問題として読む frame。適用作品は `ジョーカー_UG`。汎用化するには、笑顔を仮面、病、攻撃、解放の段階で変化させる。
- **徹底する人フレーム**: 倫理より一貫性で人物を見せる frame。適用作品は `世にも奇妙な人体実験_UG`。汎用化するには、不快な人物を善悪でなく「どこまで徹底するか」でキャラ化する。
- **ローカル運営の最適化**: 本国マニュアルより観客からの見え方を優先する frame。適用作品は `ディズニーランド_ゼミ61`。汎用化するには、キャラの正しさを設定資料でなく体験者の認知から調整する。
- **三つの大厄フレーム**: 人類史を飢餓・伝染病・戦争との戦いとして整理する frame。適用作品は `ホモデウス_UG`。汎用化するには、抽象史を「主人公と三つの敵」の構図に置く。
- **スマホ=1日2600回通えるパチンコ屋**: スマホを報酬抽選装置として擬人化・敵化する frame。適用作品は `スマホ脳_UG`。汎用化するには、依存対象を悪意ある人物ではなく誘惑する装置としてキャラ化する。
- **能力も運のうちフレーム**: 努力や才能自体も遺伝・環境に支えられると見る frame。適用作品は `能力主義_ゼミ404`。汎用化するには、成功者と敗者の人物配置を自己責任から環境差へ組み替える。
- **聞かれたことに答えない**: 表面の質問ではなく奥の不安へ答える frame。適用作品は `思考方法論_ゼミ315`。汎用化するには、相談者を「本当に聞きたいことがズレている人物」として読み直す。

## 6. 応用 PLAYBOOK (claude が動画作成時に引く)

- **泣ける人物分析を作りたい**: → 技法 #1, #2 (キャラに泣かせず、無表情と残骸で観客に感情を作らせる)
- **異界突入の説得力を作りたい**: → 技法 #5, #6 (主人公の所属を切り、境界小物を無視させる)
- **悪役を単純化したくない**: → 技法 #7, #12, #25 (合理性、管理、凡庸さを混ぜて倫理判断を揺らす)
- **非人間キャラを深くしたい**: → 技法 #13, #14, #22 (かわいさの奥に不可知性、目線のズレ、失われた由来を置く)
- **AI・人工生命の物語を作りたい**: → 技法 #13, #15 (非人間の魂より、それを扱う人間側の変化を主題にする)
- **表情で二重感情を出したい**: → 技法 #16, #26 (笑顔や名前を、幸福でなく抑圧や呪いとして使う)
- **主人公の能力を不穏にしたい**: → 技法 #17, #19 (パワーアップを怪物化、民俗記憶、歴史の返済に接続する)
- **抽象テーマを見える話にしたい**: → 技法 #27, #28 (概念を人物配置や敵の擬人化へ落とす)
- **小物だけで過去を匂わせたい**: → 技法 #9, #18, #20 (不在者、儀式、環境適応を小物と身体に預ける)
- **ブランドキャラの一貫性を作りたい**: → 技法 #29, #30 (比率と動作文法を固定し、出会うたび同じ存在に見せる)
- **解説者自身を信頼させたい**: → 技法 #31, #32 (制作者や語り手を、実務判断と癖が見える人物として置く)
- **不快な人物を扱いたい**: → 技法 #25, #33 (魅力化せず、一貫した行動原理と凡庸さで怖さを作る)
