# 物語構造

## 1. 軸の定義
この軸は、作品や解説動画の意味がどの順番で開示され、どの時点で観客の理解が反転するかを扱う。対象は起承転結、ジャンル転調、伏線回収、メタファー、アレゴリー、作中視点の入れ子、表層と深層の二重構造、神話的な型である。岡田斗司夫メソッドでは、物語構造は「あらすじ」ではなく、観客が誤読し、気づき、読み直すための認知導線として設計される。書籍解説では、論の順序、反論の置き方、具体例の回収を物語構造として扱う。

## 2. 中核原則 (Top-Level Principles)

### 原則 1: 表層ジャンルを信じさせてから深層ジャンルへ反転する
岡田の読解では、作品は最初にわかりやすい顔を出し、後からその顔を裏切る。『千と千尋』は冒頭をホラーとして始めて労働ファンタジーへ転じ、『ジョーカー』はヴィラン誕生譚に見せて孤立者の拡散構造へ向かい、『火垂るの墓』は反戦悲劇に見せて煉獄ミステリーへ変わる。講義系でも『能力主義』は魅力的な理想から始め、最後にステルス差別へ反転させる。 `[出典: 千と千尋_UG306_前編, ジョーカー_UG, 火垂るの墓_UG226, 能力主義_ゼミ404]`

### 原則 2: 冒頭は結末の縮図であり、伏線は世界の規則を先に置く
冒頭に置かれる小物、地形、建物、視線、音は、後で意味を回収するための証拠である。『火垂るの墓』の灰皿と現代夜景、『ナウシカ』の廃村と宇宙船残骸、『もののけ姫』の土面と右腕、『千と千尋』の鳥居や祠は、単なる雰囲気ではなく物語全体の設計図になる。書籍解説でも冒頭の分類や反論が後半の結論に回収される。 `[出典: 火垂るの墓_UG226, ナウシカ_開拓史編_UG, もののけ姫_UG, スマホ脳_UG]`

### 原則 3: 二重構造は「別解釈」ではなく、表層を駆動する隠れた力である
表層と深層は並列の読みではなく、表層で起きる奇妙さを深層が説明する関係にある。『かぐや姫』では女性抑圧や生命賛歌の奥に月の超常存在の罪罰があり、『トトロ』ではかわいい隣人の奥に古代神の敗北史がある。『ブレードランナー』ではレプリカント狩りの奥に、人間が非人間的行為で人間性を失う倫理がある。 `[出典: かぐや姫の物語_UG231, となりのトトロ_UG, ブレードランナー_UG, 千と千尋_UG307_後編]`

### 原則 4: 視点をずらすと善悪、救済、感動の意味が変わる
同じ出来事を誰の視点で見るかによって、物語の倫理は反転する。『火垂るの墓』では人間から見た蛍と蛍から見た兄妹が入れ替わり、『もののけ姫』では英雄の旅立ちが共同体の祟り返しにもなる。『ジョーカー』では観客が被害者に共感しつつ、排除してきた側にも置かれる。 `[出典: 火垂るの墓_ゼミ226, もののけ姫_UG, ジョーカー_UG, OnYourMark_UG311]`

### 原則 5: メタ構造は作品外の制作、媒体、社会制度を物語内へ圧縮する
岡田は作中世界を閉じた虚構として読まず、制作現場、宣伝、メディア、制度史を物語の内側に折り込む。『千と千尋』の油屋はジブリであり、『ディズニーランド』は映画セットと不動産とメディアの複合体であり、『世にも奇妙な人体実験』は医学史から都市伝説と文学史へ広がる。 `[出典: 千と千尋_UG307_後編, ディズニーランド_ゼミ61, 世にも奇妙な人体実験_UG, ブレードランナー_UG]`

### 原則 6: 解説は答えを先に置かず、誤読から再読へ導く
強い解説は「正解」を即座に出さず、一般的な受け取り方をいったん成立させてから、画面証拠や概念整理で壊す。『火垂るの墓』は泣ける悲劇という誤読を利用し、『かぐや姫』は美しい作画と女性問題を目くらましとして扱い、『思考方法論』は悩み相談への回答ではなく相談文の構造読解へ移る。 `[出典: 火垂るの墓_ゼミ226, かぐや姫の物語_UG231, 思考方法論_ゼミ315, ホモデウス_UG]`

## 4. 岡田 frame 集 (この軸で岡田が独自命名した分析 frame)

- **レイヤー構造 frame**: 解釈がレベル1、2、3と進むたびに前の読みを否定する枠。適用作品: `OnYourMark_UG311`。汎用化: 解説を「素直な読み -> 不穏な証拠 -> 深層の反転」の章立てにする。
- **悪意ある曲解 frame**: 歌詞や明るい表現を素直に映像化せず、反対の残酷な意味へずらす枠。適用作品: `OnYourMark_UG311`。汎用化: 原作テキストと画面を意図的に不一致にして二重構造を作る。
- **生クリームが美味しすぎる問題**: 表層の魅力が強すぎて本体のテーマが見えなくなる枠。適用作品: `かぐや姫の物語_UG231`, `火垂るの墓_ゼミ226`。汎用化: 感動、美術、社会問題を一度分離し、その奥の構造を再提示する。
- **貴種流離譚**: 神的存在が地上で苦労し、罪を償って元の世界へ戻る神話枠。適用作品: `かぐや姫の物語_UG231`。汎用化: 古典やファンタジーを、追放、試練、帰還、喪失の型へ接続する。
- **翁物語の乗っ取り**: 本来の主人公の喪失譚を、別人物の主観へ置き換える枠。適用作品: `かぐや姫の物語_UG231`。汎用化: 原作改変では「誰の喪失を中心にするか」を変えて意味を作る。
- **ラピュタの構造反転**: 空から来た少女を守る冒険譚を、守る側の願望が報われない喪失譚へ反転する枠。適用作品: `かぐや姫の物語_UG231`。汎用化: 成功する冒険構造を、現実的な別れの構造へ変換する。
- **ヒーロー不在の闇落ち**: 正義側の視点へ戻らず、落ちていく側の主観を追う枠。適用作品: `ジョーカー_UG`。汎用化: 転落ものでは救済役を置かず、観客を救済不能の観察者にする。
- **悪のインフルエンサー**: 悪を支配せず、周囲に悪を流行させる存在として読む枠。適用作品: `ジョーカー_UG`。汎用化: 現代的な悪役を命令者でなく模倣の発火点にする。
- **バットマン妄想説**: 正義の味方をヴィランの相手役、あるいはサイドキックとして反転して読む枠。適用作品: `ジョーカー_UG`。汎用化: 既存神話の主従関係を逆にして再解釈する。
- **冒頭5秒の謎**: 開始直後に作品全体の時間構造やルールを隠す枠。適用作品: `火垂るの墓_UG226`, `火垂るの墓_ゼミ226`。汎用化: 冒頭の一瞬を再視聴で意味が変わる設計図にする。
- **火垂るの墓はミステリー**: 悲劇ではなく、主人公がなぜ呪われるかを解く話として読む枠。適用作品: `火垂るの墓_ゼミ226`。汎用化: 既知の結末を、原因探索の構造へ変える。
- **煉獄構造**: 主人公を天国にも地獄にも行けない中間状態に置き、苦痛の反復を物語にする枠。適用作品: `火垂るの墓_UG226`。汎用化: 回想を単なる過去ではなく、現在進行形の罰として扱う。
- **文学アニメ**: キャラクターの内面を明示せず、観客に答えを出させる枠。適用作品: `火垂るの墓_UG226`, `火垂るの墓_ゼミ226`。汎用化: 台詞説明を削り、小物、編集、視線で複数解釈を起こす。
- **アシタカせっ記フレーム**: 『もののけ姫』をサンの物語ではなくアシタカ伝説として読む枠。適用作品: `もののけ姫_UG`。汎用化: タイトルに誘導されず、後世に何が伝承されるかで主人公を判定する。
- **祟り返し**: 受けた呪いを相手側へ返す共同体戦略として旅立ちを読む枠。適用作品: `もののけ姫_UG`。汎用化: 善意の送り出しを政治的処理として二重化する。
- **タイトル誤導フレーム**: 題名が観客の鑑賞視点を歪める枠。適用作品: `もののけ姫_UG`。汎用化: 宣伝文句やタイトルと、画面に残る構造を分けて読む。
- **トトロのダークサイド**: かわいい表層の背後に古代神の敗北史を読む枠。適用作品: `となりのトトロ_UG`。汎用化: 児童向け表層の下に、大人向けの歴史層を埋める。
- **理屈を隠した理屈作品**: 感覚的に見せながら、深層では論理をぎりぎりまで組む枠。適用作品: `となりのトトロ_UG`。汎用化: 子供には直感、大人には構造が見える二重設計にする。
- **宮崎作品三部循環**: 複数作品を「巨大神と人間の関係」の時代史として読む枠。適用作品: `となりのトトロ_UG`, `ナウシカ_開拓史編_UG`, `もののけ姫_UG`。汎用化: 作家作品を横断し、モチーフの歴史を組む。
