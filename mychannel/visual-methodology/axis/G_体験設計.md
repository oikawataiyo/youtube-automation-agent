# 体験設計

## 1. 軸の定義
この軸は、観客・来場者・視聴者を「見ている人」から「世界に入った人」へ変える設計を扱う。映像内の都市、異界、共同体、講演の導入、テーマパークの動線を、入口・滞在・待機・発見・退場・再訪の連続した体験として読む。

中心になるのは、ディズニーランド論にある「夢を支える打算」である。没入は雰囲気だけで成立せず、視線誘導、待ち時間、歩行速度、情報量、未完了感、運営コスト、周辺ノイズの遮断までを同時に設計した時に強くなる。

映像作品では、ブレードランナーの都市滞在、ナウシカの入村ルート、千と千尋の境界侵犯、トトロの日常と異界の交差が、この軸の応用例になる。講演や解説動画では、章立て・予告・論点棚卸し・視聴者自身の行動参加が、体験設計の代替装置になる。

## 2. 中核原則 (Top-Level Principles)

### 原則 1: 入口は説明ではなく状態変化として作る
体験は「ここから本編です」と告げるだけでは始まらない。アーケード、トンネル、鳥居、橋、廃村、山道、待機列のような境界物を通過させ、観客の身体感覚を現実から別世界へ切り替える必要がある。重要なのは、境界を一つだけ置くのではなく、複数の小さな違和感を順に踏ませること。 `[出典: ディズニーランド, 千と千尋の神隠し UG306 前編, 風の谷のナウシカ 開拓史編, となりのトトロ]`

### 原則 2: 動線は理解順であり、感情順である
来場者や観客が歩く順番は、そのまま情報を理解する順番になる。ディズニーランドは入場直後に城を遠景目標として置き、風の谷はユパの入村ルートで危険、防御、水、生活を順に見せる。直線的に最短距離で進ませるより、曲がる、待つ、見上げる、降りるという運動を入れた方が、場所の意味が体験として残る。 `[出典: ディズニーランド, 風の谷のナウシカ 開拓史編, ブレードランナー, 千と千尋の神隠し UG306 前編]`

### 原則 3: 完全消費させず、未体験を報酬化する
満足度は「全部見た」で最大化するとは限らない。ディズニーランドの80%回遊設計は、達成感と未完了感を同時に残し、再訪動機を作る。映像では、ブレードランナーの暗部や反射、ナウシカの背景装置、もののけ姫の説明されないコードが、再視聴時の報酬になる。 `[出典: ディズニーランド, ブレードランナー, 風の谷のナウシカ 開拓史編, もののけ姫]`

### 原則 4: 世界の没入は中心だけでなく境界外まで管理する
没入を壊すのは、主役の弱さだけではない。園外ホテル、安っぽい周辺景色、説明的すぎる案内、現実的な汚れ、読めすぎる地図、過度に便利な導線も世界を破る。フロリダの巨大敷地や園内BGM、酒を売らない判断、清掃の即時性は、体験の外縁を管理することで中心の夢を守る方法である。 `[出典: ディズニーランド, 千と千尋の神隠し UG306 前編, となりのトトロ, ブレードランナー]`

### 原則 5: 待つ・迷う・買う・見るを物語内行動に変える
体験設計では、待ち時間や移動時間は削るだけの無駄ではない。待機列映像は前奏になり、曲がった通路は探索になり、土産購入は記憶の回収になり、混雑は期待感にもなる。映像でも、長い移動、沈黙、街を眺める時間は、世界に滞在している感覚を作る。 `[出典: ディズニーランド, ブレードランナー, 風の谷のナウシカ 開拓史編, 千と千尋の神隠し UG307 後編]`

### 原則 6: 体験後に語れる発見を残す
良い体験は、その場で完結せず、帰った後に「実はあれは」と語れる。建物の縮尺、色彩遠近、道の曲げ方、背景インフラ、目の反射、境界物、家族の伏線のような発見は、鑑賞後の会話と再訪を発生させる。暴露ではなく、夢の作り方として語れる裏側を残すのが強い。 `[出典: ディズニーランド, ブレードランナー, 風の谷のナウシカ 開拓史編, 千と千尋の神隠し UG307 後編, もののけ姫]`

## 4. 岡田 frame 集 (この軸で岡田が独自命名した分析 frame)

- **夢と魔法と打算の国**
  - 定義: 夢の体験を支える経営・運営・錯覚の設計を読む frame。
  - 適用作品: ディズニーランド。
  - 汎用化の方法: 感動体験を雰囲気でなく、価格、待機、清掃、人流、保守、周辺環境の連鎖として分解する。
- **顧客満足度最大値化**
  - 定義: 平均滞在時間と回遊率から体験量を決める frame。
  - 適用作品: ディズニーランド。
  - 汎用化の方法: 視聴時間や来場時間に対して何%理解・達成させるかを先に決める。
- **80%回遊設計**
  - 定義: 全部体験させないことで再訪欲求を作る frame。
  - 適用作品: ディズニーランド。
  - 汎用化の方法: 未完了を不満でなく次回の理由になる位置へ残す。
- **待機時間の物語化**
  - 定義: 行列を本編前の導入に変える frame。
  - 適用作品: ディズニーランド。
  - 汎用化の方法: 待ち時間、ローディング、前説、注意事項を世界観説明や期待形成へ変える。
- **映画セット空間化**
  - 定義: 映画美術の錯覚を歩ける建築に転用する frame。
  - 適用作品: ディズニーランド。
  - 汎用化の方法: 実寸・正確さより観客視点からの大きさ、距離、奥行き、進行方向を優先する。
- **周辺土地までの没入管理**
  - 定義: 体験外の安っぽさや現実感を排除する frame。
  - 適用作品: ディズニーランド。
  - 汎用化の方法: 画面外、会場外、アプリ外、導入前後のノイズまで没入の一部として扱う。
- **ブレードランナーはオタク向けディズニーランド**
  - 定義: 物語より世界に滞在する快楽を売る frame。
  - 適用作品: ブレードランナー。
  - 汎用化の方法: 筋の強さだけでなく「入場したい場所か」を作品評価軸に入れる。
- **世界没入型映画**
  - 定義: ストーリーではなく風景を繰り返し体験する映画を読む frame。
  - 適用作品: ブレードランナー。
  - 汎用化の方法: 再視聴で発見が増える画面密度と場所の記憶を設計する。
- **ブラナウシカ**
  - 定義: 風の谷を旅行記のように歩き、地形と設備から世界史を読む frame。
  - 適用作品: 風の谷のナウシカ 開拓史編。
  - 汎用化の方法: 世界観を辞典で説明せず、入口からの観光動線で体験させる。
- **境界侵犯 frame**
  - 定義: 鳥居、祠、石畳、橋、トンネルを段階的な結界として読む frame。
  - 適用作品: 千と千尋の神隠し UG306 前編。
  - 汎用化の方法: 異界のルールを門番の台詞でなく境界物の連続で知らせる。
- **宙ぶらりん frame**
  - 定義: 所属が不安定な人物だけが異界に入れると読む frame。
  - 適用作品: 千と千尋の神隠し UG306 前編。
  - 汎用化の方法: 主人公の心理状態と世界への入場条件を結びつける。
- **観客を信じすぎた映画**
  - 定義: 分かるはずのない記号を大量に置き、難解さがリピートと議論を生む frame。
  - 適用作品: もののけ姫。
  - 汎用化の方法: 説明不足を単なる欠陥にせず、再鑑賞の報酬として成立する量に調整する。

## 6. 応用 PLAYBOOK (claude が動画作成時に引く)

- **視聴者を一気に世界へ入れたい**: → 使う技法 #1 (門、橋、音、色などの境界物を連続させ、説明前に状態を変える)
- **長い導入や待ち時間で離脱される**: → 使う技法 #5 (待機中に本編理解の前奏を置き、待たされ感を参加感へ変える)
- **解説が一本道で退屈になる**: → 使う技法 #3 (曲線導線の発想で、発見のある寄り道を章間に置く)
- **作品や場所を再訪したくさせたい**: → 使う技法 #4 (80%理解で満足させ、残り20%を次回の観察ポイントにする)
- **世界観を説明台詞に頼ってしまう**: → 使う技法 #13 (背景物に機能を持たせ、観客が用途から世界を推理できるようにする)
- **都市や室内を主役にしたい**: → 使う技法 #15 (人物の行為より、そこに滞在したいと思える光、音、湿度、雑踏を設計する)
- **異界突入を怖くしたい**: → 使う技法 #18 (怪物より先に、秩序が壊れた境界物を置いて侵犯の不安を作る)
- **初見後に語りたくさせたい**: → 使う技法 #17 (一回目では気づきにくい小物や反射を置き、発見を会話の種にする)
- **視聴者を受け身にしたくない**: → 使う技法 #24 (視聴者自身の行動を問い、解説の中で実験参加者にする)
- **ラストの余韻を強めたい**: → 使う技法 #21 (帰れない方向、閉じる扉、片道の移動を美しく見せて、説明せずに余韻を残す)
