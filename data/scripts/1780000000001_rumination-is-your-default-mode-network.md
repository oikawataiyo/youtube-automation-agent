## Section 1 – 何もしていない時に動く網

何もしない、という状態について大事なのはここだ。脳はそれを、何もない時間として扱わない。むしろ招待状として受け取る。2001年、Marcus Raichle らは PNAS に、脳機能の default mode を述べた論文を発表した。外の課題に向かっている時より、休んでいる時に強く動く、まとまりをもった脳内の網があるという話だった。

奇妙だったのは、scanner の中で脳が生きていたことではない。当然、脳は働いている。奇妙だったのは、何も課題を与えられていない時に、特定の領域が一緒に立ち上がるように見えたことだ。medial prefrontal cortex。posterior cingulate と precuneus。angular gyrus。medial temporal lobe の一部。世界があなたの注意を求めるのをやめた瞬間、別の回路が姿を見せる。

さらに2010年、Andrews-Hanna、Reidler、Sepulcre、Poulin、Buckner は Neuron で、この default network がぼんやりした一塊ではないことを示した。記憶や場面の構成に深く関わる部分もあれば、self や社会的な意味づけに関わる部分もある。つまり、休息は空白だという古い考えは間違っていた。休息とは、脳が内側の劇場へ向きを変える時間なのだ。

脳は過去を再生し、未来を仮に作り、他人が自分をどう見ているかを組み立てる。それ自体は役に立つ。時間を行き来できる心は、学び、計画し、謝り、想像し、自分を守れる。問題は loop があることではない。問題は、その loop が、いま自分のいる部屋よりも説得力を持ち始めることだ。もし default network が、ほかに何も起きていない時に話し始める仕組みだとしたら、depression はもっと冷たい問いを生む。ほかの何かが、その声を止められなくなったらどうなるのか。

## Section 2 – depression の中で loop は何をするのか

ここから、思考が巻き込まれることは性格の弱さではなく、切り替えの問題として見えてくる。2009年、Sheline らは PNAS で、depression における default mode network と self-referential processes について研究を発表した。沈み込んだ人たちは、ただ悲しい考えが普通の仕組みの中に浮かんでいるだけではなかった。default-mode 領域の活動が高く、さらに大事なことに、脳が外へ向きを変える必要がある時、その網を弱める力が落ちていた。課題は変わる。けれど網は手を離さない。

Hamilton、Farmer、Fogelman、Gotlib による2015年の Biological Psychiatry の総説は、rumination を clinical neuroscience の暗い物質の問題として扱った。本人には rumination の中身が見える。けれど、その下でどの網が支配的になっているかは見えない。本人は「考えるのを止められない」と言う。scanner は、別の網が引き継ぐべき時に、self に関わる網が起き続けていると言う。

この違いは大きい。問題が考えの中身だけなら、悪い考えをもっと良い考えに置き換えればよい。たしかに、それで助かることもある。けれど本気で rumination に飲まれた人なら、きちんと筋の通った反論を作ったのに、心がそれを無視する屈辱を知っている。別の部屋から流れる音に向かって議論しているようなものだ。文は変わるのに、流れている場所は変わらない。

depression はしばしば洞察のように感じられる。なぜなら、それは自分の声を使うからだ。分析的に聞こえる。正直に聞こえる。もう一度だけ証拠を見直そう、と言う。だが Hamilton らの2015年 Biological Psychiatry の総説が正しいなら、その loop はあなたの人生を解こうとしているだけではない。現在へ主導権を返すことに失敗している。だから「考えるのを止められない」という文は、神経学的にもかなり正確なのかもしれない。次の問いは、なぜ loop が同じ題材ばかり選ぶのかだ。

## Section 3 – なぜ loop は self に固定されるのか

ここは少し居心地が悪い。default network は、あなたのことを気にかけるから壊れているのではない。それはむしろ本来の仕事だ。2010年、Andrews-Hanna らは Neuron で、default network の機能的な分かれ方を述べた。そこには、自伝的記憶、未来の仮構、self に関わる意味づけを支える働きがある。普通の生活では、これが連続した自分を作る。傷ついた出来事を覚え、起こりうる未来を思い描き、他人の心の中にいる自分を追跡する。

けれど、人を賢くする同じ仕組みは、内側の監視装置にもなりうる。Nolen-Hoeksema、Wisco、Lyubomirsky による2008年の Perspectives on Psychological Science の総説は、rumination を、苦痛の原因と結果に対する反復的で受け身の注目と定義し、それが depression の発症、重さ、長さを予測することを示した。ここで重要なのは、受け身という言葉だ。rumination は忙しく感じられる。だが多くの場合、何かを進めてはいない。同じ材料のまま、問いの周りを回っているだけだ。

なぜあんなことを言ったのか。これを台無しにしたらどうしよう。これは自分について何を意味するのか。題材はいつも自分だ。けれど、そこで自分は思いやりを向けられているわけではない。処理されている。

さらに2010年、Killingsworth と Gilbert は Science で、数千人への経験抽出を用い、さまよう心は不幸な心であることを示した。心がそれることは、後の低い気分を予測し、その逆より強かった。これは想像が悪いという意味ではない。足場のない self の時間旅行には代償があるという意味だ。

時間の中の自分を作るための脳内の網が、自分に時間の線があること自体を罰する網になる。過去は法廷になる。未来は脅威の試算になる。他人の心は、拒絶を映す画面になる。そしてそれが内省の声で起きるため、責任感のふりをすることができる。悲劇は、loop が self を守ろうとして、self から離れられなくしていることだ。だから有効な介入は、議論に勝つことではない。loop との関係を変えることに向かう。

## Section 4 – scanner の中で何が静めるのか

では、実際に何がこうした loop を静めるのか。最初の答えは派手ではない。再現性が高い動きは、注意の訓練だ。2011年、Brewer、Worhunsky、Gray、Tang、Weber、Kober は PNAS で、熟練した瞑想者において default mode network の活動と結合が異なることを示した。瞑想中には主要な中枢の活動が下がり、休息中の結びつきにも違いが見られた。

ここが大事だ。瞑想は、前向きな考えを作る行為ではない。self の loop をひとつの出来事として気づき、呼吸、音、体、感覚へ戻る練習を何度も行うことだ。語り手を破壊するわけではない。語り手が画面全体ではないと、脳に教えている。

効果は魔法ではない。即座の悟りでもない。測ることができ、不完全で、それでも意味を持つほどには現実の変化だ。そして scanner の話は、心理学を神秘から少し引き戻してくれる。もし rumination が持続した default-mode の支配なら、self への吸収を何度も中断する練習は、生活の飾りではない。網の訓練だ。

同じ原理は、より極端で慎重に管理された研究にも現れる。2012年、Carhart-Harris らは PNAS で、psilocybin が psychedelic state における default-mode のまとまりを急性に変化させると報告した。この発見は ego dissolution や後の気分変化と並べて語られることが多い。これは、誰もが化学的な近道を追うべきだという意味ではない。狙っている場所が一貫しているという意味だ。

人が楽になる時、self-model の周囲で何かがほどける。loop は絶対ではなくなる。声はまだある。だが部屋が広くなる。そこまで見えると、実用的な問いはもっと小さく、もっと役に立つ形になる。scanner なしで、小さな解放をどう作るか。

## Section 5 – scanner なしで loop の外へ出る方法

ここからは、実際に使える部分だ。考えることを燃料にしている網を、考えで打ち負かすことはできない。だから入力を変えるか、出力への立ち位置を変える。

第一に、文法を変える。2014年、Kross らは Journal of Personality and Social Psychology で、三人称の self-talk について研究を発表した。「なぜ私はつらいのか」ではなく「なぜ Sarah はつらいのか」と問うと、一人称に浸る場合より感情の反応が下がった。こつは、自分が別人だと装うことではない。心理的な距離をほんの少し足すことだ。loop は、これは私そのものだと言う。言葉は、これは私に起きていることだと言い直す。

第二に、体を、要求のある世界へ移す。2006年、Dimidjian らは Journal of Consulting and Clinical Psychology で、大きな depression 試験において behavioral activation が cognitive therapy や antidepressant medication と強く競り合ったことを報告した。行動が rumination と神経上の場所を取り合うと考えれば、これは筋が通る。散歩、入浴、電話、皿洗い、仕事の一頁。外から見ればばかばかしいほど小さくても、内側では生物学的に意味がある。task-positive system が握るものを与えるからだ。

第三に、loop に名前をつける。だが交渉しない。「自分の価値について最終真理を発見した」ではない。「default network が拒絶の記録を再生している」くらいでいい。わざと臨床的に聞こえるようにする。啓示に見えたものを、出来事へ戻すためだ。

第四に、問題が自分全体である時は、ひとりで問題解決をしすぎない。問いが実務的なら、次の行動を書く。問いが午前二時の存在論なら、夜明けまで置いておく。

もちろん、これらは万能の治療ではない。depression には治療、支え、時には薬が必要だ。ただ、動作原理は単純だ。固まった音量つまみとは議論しない。その周りの条件を調整する。すると、loop の終わりが意味できることも変わってくる。
